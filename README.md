#期中專案點餐機<br>

##簡介<br>
利用課堂中所學，自己選定一個點餐系統作為參考模板，用程式編寫出類似應用之作業。此點餐系統以五桐號線上訂餐作參考源，用戶可以選擇飲料類型、甜度、冰塊、優惠方案、支付方式和發票開立方式。<br>

##功能<br>
選擇飲料類型<br>
選擇甜度和冰塊<br>
確認訂單並顯示價格
選擇優惠方案<br>
選擇支付方式（現金、Line pay）<br>
選擇發票開立方式（紙本、手機載具、捐贈）<br>
完成訂單<br>
